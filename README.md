# 3D 시각장애인 보행 체험 시뮬레이션 & 배리어 프리 캠페인
> **"빛을 잃은 보도 위, 노란 선을 따라 걷다"**

이 프로젝트는 시각장애인의 보행 환경을 직접 3D로 체험함으로써 인도 위 불법 주정차와 파손된 점자블록이 시각장애인에게 미치는 실질적인 위협을 알리고 공감대를 형성하기 위해 기획된 **3D 웹 시뮬레이터 및 배리어 프리(Barrier-Free) 캠페인 사이트**입니다.

## 🔗 프로젝트 특징
- **웹 브라우저 즉시 실행**: 무거운 설치 없이 크롬 등 모던 웹 브라우저에서 HTML5, Three.js 기술을 기반으로 즉시 구동됩니다.
- **몰입형 다크 UI**: 고급스러운 글래스모피즘(Glassmorphism)과 황색 점자블록을 강조한 다크 테마 포트폴리오 사이트 내에 시뮬레이터를 통합 구현했습니다.
- **실감 나는 오디오 피드백**: 헤드폰을 착용하면 입체 비프음과 바닥 재질별 지팡이 진동 타격음을 통해 시각 제한을 극복하는 공감 체험을 극대화합니다.

---

## 🎮 시뮬레이션 주요 기능

1. **3D 몰입형 보행 환경 (Three.js)**
   - 가운데 황색 점선과 양쪽 실선이 그려진 2차선 도로 및 횡단보도.
   - 보도의 중심을 따라 깔려 있는 황색 점자블록(유도용 선형 블록 & 대기용 점형 블록).
   - 주변 빌딩숲, 가로수, Spotlight가 내리쬐는 가로등 배치로 현장감 넘치는 야간 도시 보도 구현.

2. **시각 제한 페이딩 (암전 효과)**
   - 평소 점자블록 위로 걸을 때는 시야가 올바르게 보입니다.
   - 전방의 **파손된 점자블록**을 밟거나, 점자블록을 **벗어나 인도 또는 차도로 진입**하면 시야가 약 99.9% 어두워지는(거의 보이지 않는 암전) 리얼한 비주얼 제한을 경험합니다.

3. **배리어 프리 음향 신호기 시스템**
   - 횡단보도 양쪽에 설치된 2개의 세로형 신호등(적색 5초 ➡️ 황색 1.5초 ➡️ 녹색 5초 주기).
   - **음성 안내 버튼 상호작용**: 평소에는 신호등 불빛이 흑백으로 보이다가, 신호기 기둥 근처에서 **E 키**를 누르거나 마우스로 버튼을 클릭하면 음성 안내 방송(*"초록불입니다. 건너가도 좋습니다."*)과 함께 신호가 컬러로 밝혀집니다.
   - 녹색/황색/적색 신호에 맞추어 주기가 달라지는 입체 안내 비프음 출력.

4. **지팡이 촉각 사운드 (Web Audio API)**
   - 주인공인 시각장애인 시점에서 흰 지팡이를 들고 보행합니다.
   - 밟고 있는 바닥 재질에 따라 실시간으로 지팡이 타격음이 다르게 울립니다:
     - **황색 점자블록**: 경쾌하고 맑은 "탁!" 소리
     - **일반 인도**: 단단한 보도블록 "툭!" 소리
     - **깨지거나 부서진 점자블록**: 덜컹거리는 파손 타일 소리
     - **아스팔트 도로**: 둔탁하고 묵직한 소리

5. **현실적인 위험 요소 및 게임 오버**
   - **무단횡단 사고**: 신호등이 **빨간불(적색)**일 때 차도에 진입할 경우 즉시 차량 충돌 게임 오버가 발생합니다.
   - **불법 주정차 충돌**: 점자블록 위에 불법 주차된 방해 차량에 3번 부딪힐 경우 보행 불능 상태가 되어 게임 오버됩니다. (가까워지면 소리 및 시각 경고 배너 발생)
   - **다시 시작**: 게임 오버 시 모달창의 "다시 시작(Restart)" 버튼을 누르면 위치 및 사고 횟수가 리셋되며 다시 마우스 락이 적용됩니다.

## ⌨️ 조작 방법

- **이동**: `W` `A` `S` `D` 키 (1인칭 시점 이동)
- **시야 회전**: 마우스 움직임 (화면 클릭 시 마우스 잠금 활성화)
- **일시 정지 / 마우스 해제**: `ESC` 키 (일시정지 화면에서 다시 화면을 클릭하면 재개)
- **신호기 작동**: 기둥 근처에서 `E` 키 또는 기둥의 버튼 클릭

## 🛠️ 실행 방법 (로컬 호스팅)

웹 오디오 API 및 TTS(음성 안내) 서비스는 브라우저의 보안 정책상 파일 프로토콜(`file:///`)에서는 일부 제한될 수 있으므로, 아래와 같이 로컬 서버를 통해 구동하는 것을 권장합니다.

1. 터미널을 열고 프로젝트 폴더 경로로 이동합니다:
   ```bash
   cd "C:\Users\master\Desktop\동행 프로젝트"
   ```
2. 포함된 PowerShell 가벼운 HTTP 서버 스크립트를 실행합니다:
   ```powershell
   powershell -ExecutionPolicy Bypass -File server.ps1
   ```
3. 브라우저를 열고 아래 주소로 접속합니다:
   ```
   http://127.0.0.1:8000/
   ```
4. 소개 페이지를 읽어본 뒤 **[체험하기]**를 눌러 시뮬레이션을 즐겨보세요!
